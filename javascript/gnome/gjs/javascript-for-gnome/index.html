JavaScript for Gnome

By Valera Rozuvan December 23, 2013 Permalink source of post

I have just discovered about the coolest thing ever since I last discovered the Gnome desktop. It is Gjs: Javascript Bindings for GNOME.

What is Gjs? Quoting from the home page

Gjs is a JavaScript binding for GNOME. It’s mainly based on Spidermonkey JavaScript engine and the GObject introspection framework.

I have already set up a GitHub repository to mess around with it.
